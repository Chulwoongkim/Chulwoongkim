https://worship.ichungeoram.com/

https://slides.com/
를 이용하시면 쉽게 만들 수 있습니다. ppt 만들듯이 슬라이드를 만들고, 주소를 배부하면 됩니다.

직접 코딩을 하실 분들은 
reveal.js 를 이용하시면 더욱 보기 좋은 내용을 만드실 수 있습니다.
